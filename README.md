# Configuration 
My work configuration for Unix-like systems, specifically Debian 11.

Tools in use
------------
  * Bash (Shell)    
  * Vim (Text Editor)    
  * uxterm (Terminal App)
  * tmux (Multiplexer)
  * Git (Version Control)    
  * SSH (Remote Access)

Install
-------
```
./setup.sh
```

Please Note
-----------
Installation script does not backup files that are overwritten.
Backup is recommended before installation.
