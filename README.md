# Configuration 
My work configuration for Unix-like systems.    

Tools in use
------------

  * Vim (Text Editor)    
  * Git (Version Control)    
  * SSH (Remote Access)

Install
-------
```
chmod +x setup.sh
./setup.sh
```

Please Note
-----------
Installation script does not backup files that are overwritten.
Backup is recommended before installation.
